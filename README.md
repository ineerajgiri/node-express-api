# Node Express API

This is my first backend API built using Node.js and Express.

## Features
- Express server
- Multiple routes
- JSON API

## Routes

| Route | Description |
|------|-------------|
| / | Home page |
| /about | About page |
| /contact | Contact page |
| /api | Returns JSON data |

## Run Locally

Install dependencies

npm install

## Run server

node app.js
